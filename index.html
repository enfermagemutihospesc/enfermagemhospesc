⚙️ Configure o Firebase para ativar a nuvem

Para salvar os dados na nuvem e compartilhar entre todos os enfermeiros, você precisa configurar um projeto Firebase gratuito.

Passo a passo:

1. Acesse console.firebase.google.com e crie um projeto

2. Ative Firestore Database (modo de produção)

3. Ative Authentication → E-mail/senha

4. Vá em Configurações do projeto → Seus apps → App da Web

5. Copie as configurações e cole no início deste arquivo HTML onde está FIREBASE_CONFIG

6. Cadastre os enfermeiros em Authentication → Usuários

Enquanto isso, o sistema funciona offline com dados locais.
